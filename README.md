# 🚀 Nesmesis Trade Testnet Autobot
---

## ⚙️ Requirements

 install:

- Node.js (v16 / v18 recommended)
- npm 
- Git

---

## 📥 Clone Repository

```bash
git clone https://github.com/v325-max/Nesmesis-Trade-Testnet-Autobot.git
cd Nesmesis-Trade-Testnet-Autobot
```

## Install Dependencies

```bash
npm install
```

## Privatekey Import
edit pk.txt, peplace with yours


## Run

```bash
node index.js
```
